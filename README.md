# miPrimerProyecto

## Requisitos
- Node.js 18+
- MongoDB en `mongodb://localhost:27017/local`

## Instalación

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

El frontend se sirve en `http://localhost:4200` y espera que el backend esté en `http://localhost:3000`.
